Tipos de datos primitivos en java

Tipo Int

Es un tipo de datos de 32 bits con signo.
Su rango va entre -2^31 hasta 2^31-1.

Tipo float

Es un tipo de datos para almacenar numeros con coma flotante de precision de 64 bits
